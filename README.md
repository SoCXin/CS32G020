## [CH567描述](https://github.com/sochub/CH567/wiki) 

[CH567](https://github.com/sochub/CH567) 是一款高性能32位精简指令集微控制器，系统主频可达120MHZ。片上集成两组独立的高速USB2.0主/从控制器、4组SDIO控制器、加解密算法模块、4组串口、7路PWM、3组定时器等丰富的外设资源，LQFP48封装可广泛应用于各种嵌入式设备。
